> ⚠️ **免责声明**：本仓库仅用于协议分析与学习研究，所有账号、密码由使用者自行输入，请勿提交或硬编码任何隐私数据。使用本项目时请遵守目标网站的使用协议和相关法律法规。

## 功能概览
- `maomi_spider.py`
  - 复现猫咪 VIP 登录的 `encode_sign` + AES-CBC 加密流程。
  - 拉取分类（含专题 topic）并分页抓取视频列表，输出结构化 JSON。
- `web_app.py`
  - Flask 单文件 Web 控制台，提供账号输入、分类加载、分页采集、专题信息展示。
  - 视频卡片展示核心元数据，点击"详情(JSON)"即可在弹窗中查看完整字段。
  - 不再提供缩略图代理或视频下载逻辑，避免隐私与版权风险。

## 快速开始
1. 安装依赖：
   ```bash
   pip install flask requests pycryptodome
   ```
2. 运行 CLI（示例）：
   ```bash
   python maomi_spider.py --username <你的账号> --password <你的密码> --category mmtj --pages 3 --output videos.json
   ```
3. 运行 Web 控制台：
   ```bash
   python web_app.py
   ```
   打开 `http://127.0.0.1:5000/`，依次输入账号、密码，点击"加载分类"→选择分类→设置页数→"开始采集"。

## Web 控制台交互
- **专题信息**：若分类为专题，会在顶部展示标题、描述、售价与外链。
- **视频卡片**：列出 ID、标签、时长、可用流（MP4/HLS），并提供"原站页面""详情(JSON)"按钮。
- **详情弹窗**：格式化输出完整 JSON，方便复制或二次分析。
- **调试日志**：底部日志区域记录每一步调用状态，便于复现与排查。
